# Security Policy

## Supported Versions

This repository is a template of AI agent configurations — it ships no runtime code, only Markdown configuration and documentation. The latest state of the `main` branch is the only supported version.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅        |

## Reporting a Vulnerability

If you discover a security issue — for example, a prompt or instruction that could cause an AI agent to take unsafe actions (leaking secrets, destructive commands, prompt injection vectors) in projects that adopt this template — please report it privately:

- **Preferred:** Use [GitHub private vulnerability reporting](https://github.com/nondito-soft/laravel-ai-agents/security/advisories/new).
- **Email:** arnab@nonditosoft.com

**Please do not open a public issue for security reports.**

### What to Include

- A description of the issue and which file(s) it affects.
- The scenario in which it could be exploited (which AI tool, what user action triggers it).
- A suggested fix, if you have one.

### What to Expect

- An acknowledgment within **72 hours**.
- A status update within **7 days** of acknowledgment.
- Credit in the fix's release notes, unless you prefer to remain anonymous.
